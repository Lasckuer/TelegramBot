# 💸 Personal Finance Tracker (aiogram 3)

## Описание
Продвинутый Telegram-бот для управления личными финансами[cite: 18]. Бот позволяет записывать расходы и доходы в Google Таблицы, сканировать QR-коды чеков, учитывать долги и анализировать траты с помощью графиков[cite: 18].

## ✨ Ключевые функции
- **Интеграция с Google Sheets**: Полная синхронизация всех транзакций с облачной таблицей[cite: 18].
- **Сканер QR-кодов**: Автоматическое распознавание чеков и добавление товаров через API ФНС[cite: 18].
- **Мультивалютность**: Распознавание символов `$` или `USD` и автоматическая конвертация по актуальному курсу[cite: 18].
- **Трекер долгов и подписок**: Учет выданных средств и напоминания о регулярных платежах[cite: 18].
- **Наглядная аналитика**: Генерация круговых диаграмм и детальное сравнение расходов с предыдущим месяцем[cite: 18].
- **Безопасность**: Middleware-фильтр для ограничения доступа к боту по Telegram ID[cite: 18].

## 🛠 Технологический стек
- **Python** 3.10+[cite: 18]
- **aiogram** 3.x (Асинхронный фреймворк для Telegram)[cite: 18]
- **Google Sheets API** (gspread)[cite: 18]
- **Pandas & Matplotlib** (Анализ данных и графики)[cite: 18]
- **APScheduler** (Планировщик задач)[cite: 18]

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone [https://github.com/Lasckuer/TelegramBot.git](https://github.com/Lasckuer/TelegramBot.git)
cd TelegramBot
```

### 2. Настройка переменных окружения
Создайте файл `.env` в корневой директории и заполните его по шаблону:
```env
BOT_TOKEN=ваш_токен_от_BotFather
SPREADSHEET_ID=айди_гугл_таблицы
USER_ID=ваш_телеграм_айди
PROVERKA_CHEKA_TOKEN=токен_от_proverkacheka_com
JSON_KEYFILE=service_account.json
```
> Также поместите ваш файл секретного ключа `service_account.json` в корень проекта[cite: 18].

### 3. Настройка виртуального окружения и зависимостей
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 4. Запуск бота
```bash
python main.py
```
