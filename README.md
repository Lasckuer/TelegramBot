# 💸 Personal Finance Tracker (aiogram 3)

## Описание
Продвинутый Telegram-бот для управления личными финансами. Бот позволяет записывать расходы и доходы в Google Таблицы, сканировать QR-коды чеков, учитывать долги и анализировать траты с помощью графиков.

## ✨ Ключевые функции
- **Интеграция с Google Sheets**: Полная синхронизация всех транзакций с облачной таблицей.
- **Сканер QR-кодов**: Автоматическое распознавание чеков и добавление товаров через API ФНС.
- **Мультивалютность**: Распознавание символов `$` или `USD` и автоматическая конвертация по актуальному курсу.
- **Трекер долгов и подписок**: Учет выданных средств и напоминания о регулярных платежах.
- **Наглядная аналитика**: Генерация круговых диаграмм и детальное сравнение расходов с предыдущим месяцем.
- **Безопасность**: Middleware-фильтр для ограничения доступа к боту по Telegram ID.

## 🛠 Используеться
- **Python** 3.10+
- **aiogram** 3.x (Асинхронный фреймворк для Telegram)
- **Google Sheets API** (gspread)
- **Pandas & Matplotlib** (Анализ данных и графики)
- **APScheduler** (Планировщик задач)

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone [https://github.com/Lasckuer/TelegramBot.git](https://github.com/Lasckuer/TelegramBot.git)
cd TelegramBot
cp env.example .env
```

### 2. Настройка переменных окружения
Изменить файл `.env` в корневой директории и заполните его по шаблону:
```env
BOT_TOKEN=ваш_токен_от_BotFather
SPREADSHEET_ID=айди_гугл_таблицы
USER_ID=ваш_телеграм_айди
PROVERKA_CHEKA_TOKEN=токен_от_proverkacheka_com
JSON_KEYFILE=service_account.json
```
> Также поместите ваш файл секретного ключа `service_account.json` в корень проекта.

### 3. 📖 Документация
Подробные инструкции по настройке проекта вынесены в отдельные файлы:
* [Инструкция по настройке Google API](docs/google_setup.md) — как создать сервисный аккаунт и подключить таблицы.
* [Работа со сканером чеков](docs/receipts.md) — как получить токен и правильно фотографировать чеки.

### 4. Настройка виртуального окружения и зависимостей
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 5. Запуск бота
```bash
python main.py
```
