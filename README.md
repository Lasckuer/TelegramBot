# 💸 Personal Finance Tracker (SQLite Edition)

## 📝 Описание
Асинхронный Telegram-бот на базе **aiogram 3.x** для управления личными финансами. В данной ветке проект переведен на локальное хранилище **SQLite**, что обеспечивает мгновенный отклик и работу без внешних зависимостей от облачных сервисов.

## ✨ Ключевые функции
- **Локальная база данных**: Полный учет транзакций в файле `finance_bot.db`.
- **Сканер QR-кодов**: Автоматический парсинг чеков через API `proverkacheka.com` и сохранение товаров.
- **Мультивалютность**: Распознавание `$` и `USD` при вводе суммы с автоматической конвертацией в рубли по текущему курсу.
- **Управление лимитами**: Установка месячного лимита трат с уведомлениями о превышении.
- **Экспорт данных**: Генерация отчетов в формате **Excel (.xlsx)** за выбранный период через встроенный календарь.
- **Аналитика и графики**: Визуализация расходов в виде круговых диаграмм и сравнение текущего месяца с предыдущим.
- **Напоминания**: Учет активных долгов и подписок с ежедневными уведомлениями в заданные даты.

## 🛠 Технологический стек
- **Python** 3.14+.
- **aiogram** 3.x (Асинхронный Telegram API).
- **SQLite3** (Локальное хранение данных).
- **Pandas** (Анализ данных и экспорт в Excel).
- **Matplotlib** (Генерация графиков аналитики).
- **Pyzbar** (Декодирование QR-кодов).

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone [https://github.com/Lasckuer/TelegramBot.git](https://github.com/Lasckuer/TelegramBot.git)
cd TelegramBot
git checkout SQLite-db
```

### 2. Настройка окружения
Скопируйте пример файла конфигурации и заполните его:
```bash
cp .env.example .env
```
В файле `.env` укажите:
- `BOT_TOKEN`: Токен вашего бота от BotFather.
- `PROVERKA_CHEKA_TOKEN`: Токен от сервиса proverkacheka.com.

### 3. Установка зависимостей
```bash
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
# venv\Scripts\activate   # Для Windows
pip install -r requirements.txt
```

### 4. Запуск тестов
Перед запуском рекомендуется убедиться в корректности работы всех модулей:
```bash
python -m pytest
```

### 5. Запуск бота
```bash
python main.py
```

## 📂 Структура проекта
```text
├── app/
│   ├── database/        # Логика SQLite (db_manager.py)
│   ├── handlers/        # Обработчики сообщений и состояний
│   ├── keyboards/       # Inline и Reply клавиатуры
│   ├── states.py        # Описание состояний FSM
│   ├── jobs.py          # Фоновые задачи (уведомления)
│   └── utils.py         # Вспомогательные функции
├── tests/               # Автоматизированные pytest-тесты
├── .env                 # Конфигурация (BOT_TOKEN и др.)
└── main.py              # Точка входа в приложение
```
