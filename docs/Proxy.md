# 📁 Гайд по получение и настройке прокси для Telegram-бота

---

## 1. 🌐 ПРОКСИ: ПОЛУЧЕНИЕ И НАСТРОЙКА

### ⚠️ Важное предупреждение (Перед покупкой)
*   **Сервер в РФ:** Прокси строго необходим. Без него бот не сможет «достучаться» до серверов Telegram.
*   **Сервер за рубежом:** Прокси **не нужен**. Трафик пойдет напрямую.
*   **Локальный ПК/Ноутбук:** Если у тебя включен системный VPN, прокси можно не настраивать, но для стабильности Docker-контейнера его всё равно рекомендуется указать.

### 🛒 Покупка прокси
1.  **Выбор сайта:** В примере используется [Proxy.Market](https://ru.dashboard.proxy.market), но подойдет любой аналогичный сервис.
2.  **Геолокация:** Покупай прокси **любой страны, кроме РФ**. Оптимально брать Европу (Германия, Нидерланды) для минимальной задержки.
3.  **Тип прокси:**
    - **IPv4 Shared (Общие):** Дешево и достаточно для бота.
    - **Протокол:** Выбирай SOCKS5 (рекомендуется) или HTTP.
3.  **Данные после покупки:** Тебе выдадут список в формате: `IP, Port, Login, Password`.

### ⚙️ Настройка в проекте (.env)
Собери данные в одну строку в файле `.env`:
`PROXY_URL=socks5://логин:пароль@IP:порт`

**Пример (если данные IP: 1.2.3.4, Port: 8080, Login: user, Pass: 123):**

```PROXY_URL=socks5://user:123@1.2.3.4:8080```

Если же у вас http то просто вмсето socks5 поменять на http
