# 🧾 Настройка сканера чеков

Для того чтобы бот мог автоматически распознавать товары из ваших чеков, необходимо подключить API сервиса «Проверка чека».

### 🚀 Пошаговая инструкция

1. **Регистрация**  
   Перейдите на сайт [proverkacheka.com](https://proverkacheka.com) и создайте личный аккаунт.

2. **Получение токена**  
   - Зайдите в свой **Личный кабинет**.
   - Перейдите в раздел **Профиль**.
   - Найдите пункт **«Токен доступа к API»**.
   - Нажмите кнопку **«Получить»** (или «Обновить»), чтобы сгенерировать уникальный ключ.

3. **Настройка бота**  
   - Откройте файл `.env` в корневой папке вашего проекта.
   - Найдите строку `PROVERKA_CHEKA_TOKEN=`.
   - Вставьте ваш полученный токен сразу после знака равно без пробелов:
     ```env
     PROVERKA_CHEKA_TOKEN=ваш_длинный_токен_здесь
     ```

### 💡 Важные примечания
* **Лимиты**: Бесплатный тариф имеет ограничения по количеству запросов в сутки.
* **Качество фото**: Для успешного распознавания фотографируйте QR-код при хорошем освещении и без сильных бликов.
* **Безопасность**: Никогда не передавайте этот токен третьим лицам и не выкладывайте его в открытый доступ (он автоматически игнорируется Git благодаря вашему `.gitignore`).
