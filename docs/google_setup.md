# 📊 Настройка интеграции с Google Sheets

Эта инструкция поможет вам связать Telegram-бота с вашей Google Таблицей для автоматического учета финансов.

---

### 1️⃣ Создание проекта в Google Cloud
1. Перейдите в [Google Cloud Console](https://console.cloud.google.com/).
2. Нажмите на выпадающее меню в левом верхнем углу (выбор проекта) и выберите **"New Project"**.
3. Назовите его, например, `Telegram-Finance-Bot`, и нажмите **"Create"**.

### 2️⃣ Включение необходимых API
Чтобы бот мог взаимодействовать с таблицами, нужно активировать два сервиса:
1. В строке поиска сверху введите **"Google Sheets API"**, выберите его и нажмите **"Enable"** (Включить).
2. Снова перейдите в поиск, введите **"Google Drive API"**, выберите его и также нажмите **"Enable"**.

### 3️⃣ Создание Сервисного Аккаунта
Это создание «виртуального пользователя», от имени которого бот будет вносить записи.
1. Перейдите в раздел **APIs & Services > Credentials** (Учетные данные).
2. Нажмите **"+ Create Credentials"** вверху и выберите **"Service Account"**.
3. Введите имя (например, `bot-manager`) и нажмите **"Create and Continue"**.
4. В поле выбора роли (**Role**) выберите **Basic > Editor** (Редактор). Нажмите **"Continue"**, а затем **"Done"**.

### 4️⃣ Скачивание JSON-ключа
1. В списке **"Service Accounts"** нажмите на email только что созданного аккаунта.
2. Перейдите на вкладку **"Keys"** (Ключи).
3. Нажмите **"Add Key" > "Create new key"**.
4. Выберите формат **JSON** и нажмите **"Create"**.
5. **Важно:** Переименуйте скачанный файл в `service_account.json` и переместите его в корневую папку с кодом бота.

### 5️⃣ Подготовка и настройка таблицы
1. Создайте новую таблицу в [Google Sheets](https://sheets.new).
2. Заполните первый лист:
   * `A1: Категория`, `B1: Название`, `C1: Стоимость`, `D1: Магазин`, `E1: Дата`.
3.Создайте второй лист и переменуйте его в "Доходы"
   * `A1: Источник`, `B1: Описание`, `C1: Сумма`, `D1: Дата`.
4. **Дайте боту доступ:**
   * Откройте файл `service_account.json` и скопируйте значение из поля `"client_email"`.
   * В Google Таблице нажмите **"Share"** (Поделиться).
   * Вставьте скопированный email и назначьте ему роль **Editor** (Редактор).

### 6️⃣ Получение Spreadsheet ID
ID таблицы находится в адресной строке вашего браузера:
`https://docs.google.com/spreadsheets/d/` **ТУТ_ВАШ_ID** `/edit`

Скопируйте этот набор символов и вставьте его в ваш файл `.env`:
```env
SPREADSHEET_ID=ваш_айди_таблицы
```

---

> **⚠️ Внимание:** Никогда не выкладывайте файл `service_account.json` в открытый доступ! Он автоматически добавлен в ваш `.gitignore` для безопасности.
